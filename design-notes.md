# Design Reference Notes

## New Wheel (newwheel.net)
- Clean, minimal navigation with clear categories
- Large hero images/videos that fill viewport
- Very little text in hero - just a headline and CTA
- White/light backgrounds with plenty of whitespace
- Simple category cards with clean images
- "INSPIRATION" label above headline - emotional appeal
- Clear CTAs: "LEARN MORE", "WATCH THE VIDEO", "TAKE A TEST RIDE"
- Calm color palette - mostly neutrals with accent color
- Product categories displayed as simple image cards
- Location info at bottom, not overwhelming
- Overall feel: calm, easy, approachable, premium

## Key Takeaways for Let It Ride:
1. Reduce text in hero section dramatically
2. Use full-width imagery
3. Keep CTAs simple and clear
4. Remove informational/educational sections from homepage
5. Focus on experience and emotion, not features
6. More whitespace between sections
7. Simpler navigation structure

## Visit Bend (visitbend.com)
- Giant typography as hero element ("BEND" fills screen)
- Full-bleed imagery with minimal text overlay
- Very minimal navigation - just "MENU" button
- Quick action buttons for common needs (gear rentals, bike shops, etc.)
- Card-based content sections
- Clean category links with simple labels
- Event listings with dates
- Journal/blog section with read times
- Overall feel: bold, adventurous, but still clean and easy to navigate

## Combined Design Direction for Let It Ride:
1. Simplify hero to large imagery + minimal text
2. Remove educational/informational sections from homepage
3. Keep only essential CTAs (Book Tour, Rent Bike, Shop)
4. Use more whitespace between sections
5. Focus on experience/emotion over features
6. Clean category cards for tours/rentals/shop
7. Move eco-savings content to About or dedicated page
